# Research

本目录预留给研究、预研和历史背景材料。

这类文档的用途是解释“为什么曾经这样想”，而不是定义“当前系统现在怎么做”。

使用规则：

- 可以保留早期分析、路线探索和外部调研
- 不作为当前实现行为的首要真相源
- 如与当前事实冲突，以 `docs/current/*`、[README.md](../../README.md) 和 [state.md](../../state.md) 为准
